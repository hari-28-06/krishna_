Overview

This project introduces a smart surveillance system that uses facial recognition technology to identify missing persons and track lost items. Integrated with communication tools and alert systems, it aims to provide real-time assistance to authorities and families in need.

Smart Surveillance System Photos

Key Features

  • Live camera feed with face recognition capabilities
    • Real-time video with OpenCV and face detection.
  • Secure database for storing known/missing persons
    • Encrypted MySQL database for secure data handling.
  • Automatic alert system via email/SMS when a match is found
    • Alerts are triggered and sent via email or SMS instantly.
  • Admin dashboard for managing users, reports, and system settings
    • Admins manage all users, reports, and view logs.
  • Data encryption and access control for privacy
    • Data protected via SSL and user roles.
